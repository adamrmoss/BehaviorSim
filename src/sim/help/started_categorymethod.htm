Creating a category method
1. Open the category definition window, by clicking the menu item "Application->Define Category", or the corresponding icon in the tool bar. Both methods are shown in the following figure:



2. Cateogry Definition Window is shown below. The left-side tree displays all defined categories.


3. Navigate to the methods node of the desired category, and right-click the node, a popup menu will shown as follows.


4. Click "Create New Method", a method editor will be shown below.


5. Type in the method in the text area and click the "Save" button. Messages of creating a new method such as "The method is created sucessfully!" will be shown in the Messages area. A new method will be created and its name will be shown in the left-side tree, once the method is created successfully. Otherwise, the errors of syntax or semantics will be shown in the popup window. A sample syntax error is shown below.

The error can be reported to BehaviorSim development group by clicking "Report Bugs" button.

The usage of method creator will be described in detail in "Using method editor".
