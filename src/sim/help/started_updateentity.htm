Update entity propertie
Once an entity is created (Refer to page "Creating a new entity" for more details), its properties (Display, Position, Fields, and Behavior Network) will be shown in the Application Navigation Tree. Each property has a popup menu associated with it. The popup menu can be shown by right clicking corresponding tree node.

1 To update Display,
        a) Right click Display node and select "Edit Display...".
        b) A Diaply editor window will be shown (see the following figure). Edit the desired properties and click "Save" button.
        Hint: Image is the full-path of the entity image, which is same as that of category when an entity is created.
        You can let the display size of the entity same as the image size by clicking the checkbox. Or, you can specify a specific display size by entering the two textfields (left textfield is width and the other one is height).




2 To update Position, there are two ways. in the simulation world, you can simply drag the entity to the desired position, OR
        a) Right click Position node and select "Edit Position...".
        b) A Position editor window will be shown (see the following figure). Edit the desired properties and click "OK" button.
        Hint: x and y represents the position of x and y dimension respectively.



3 To update Fields,
        a) Right click Fields node and select "Edit Fields...".
        b) A Fields editor window will be shown (see the following figure). Edit the desired properties and click "Save" button.
        Hint: For each row, both the field name and its value are shown.



4 To update Behavior Network, see the page "Creating behavior network" for more details.
