Creating a category
1. Open the category definition window, by clicking the menu item "Application->Define Category", or the corresponding icon in the tool bar. Both methods are shown in the following figure:



2. Cateogry Definition Window is shown below. The left-side tree displays all defined categories.


3. A new category can be created by right-clicking the tree node "Categories". Then click the item "Create New Category" from the popup menu, which brings up the category basic information definition panel in the right side. The window is shown in the following figure:


4. In the panel, Category name is used for further reference in the entity definition stage. Category picture is used for the representation purpose. The check box "Need to setup behavior network" if is checked (selected), a behavior network can later be defined for the entities of this category in the behavior network definition stage. Category fields table is used to define fields for the category. A field can be added/edited/deleted from the category by check the checkbox in the first column of each row first and then click the button "Add", "Edit", or "Delete". When the button "Add" is clicked, the following figure will be displayed:



5. In the above figure, all fields are needed to fill out and for each property, a default value should be specified. For the number type, a decimal value, for example 0.0, -1.0, should be specified. For String type, it has a value as double-quoted characters, for example, "a", "test". And the object type is not supported in the current version.

6. The final category definition can be saved in the system by clicking the "Apply" button in the category definition panel. The created category will appear in the left-side tree and application editor.
