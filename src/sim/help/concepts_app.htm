Application
In BehaviorSim, each behavior-based control simulation is defined as an Application. Three important components
of an application are Application Directory, Resource Directory, and Application Name. Application Directory is the
directory where an application is stored. Resource Directory is the directory where all resources of an application are
stored. Application Name is the name of an application.

One important file storing in the Application Directory is Model File, which will be introduced in another page. In current
version (1.0), for simplicity, Resource Directory is a directory within the Application Directory. Resource Directory can be
same as Application Directory, which means that all application data will be stored in Application Directory. Two general
directory structures of an application are shown in the following figure.

Resource Directory in the top application is same as Application Directory. While Resource Directory in the bottom application
