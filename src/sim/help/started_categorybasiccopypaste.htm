Creating a category by copy-and-paste
1. Open the category definition window, by clicking the menu item "Application->Define Category", or the corresponding icon in the tool bar. Both methods are shown in the following figure:



2. Cateogry Definition Window is shown below. The left-side tree displays all defined categories. Right-click the desired category and select "Copy" menu item.


3. In the same window, right click "Categories" node and select "Paste".


4. A new category will be created and shown in the tree.
