Best Practices
1. Always build application incremently.


BehaivorSim is still in development, which means that the software may contain some unkown bugs which may force you restart the environment in some situations. Thus, it is better to develop the application incremently and save it frequently so that you do not need to redo some work in the case of restart.

A good way to automatically save the work is using the "Automatic Save" tool in the tool bar. Once you specify time interval and click "Schedule" button, BehaviorSim will periodically save your work .

2. Always follow the instructions from the help documentation.

3. Send us email by clicking "Report Bugs" button in the popup exception window.
