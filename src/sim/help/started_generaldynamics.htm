Creating general dynamics
1 Bring up the window for creating general dynamics. There are four ways to do this.
1). Double click the "Entity Dynamics" node in the navigation tree.
2). Right click the entity node and in the popup menu, select "Setup Dynamics" -> "General Dynamics".
3). Select the entity node and click the "Define Entity Dynamics" item in the tool bar.
4). Select the entity node and click the "Define Entity Dynamics" item in the menu bar.

The following figure shows the approach 2), 3), 4).


The interface of creating general dynamics is shown below.


2. Type in Java code representing the general dynamics.
Note that, you can type the character '@' to see all available system APIs, and self-defined methods or fields. The methods and fields are shown in a popup list as below. The methods and fields are categorized and sorted. Both keyboard and mouse can be used to select an item. See the page "Using method editor" for more details.

3. Click the button "OK" to set the general dynamics.
